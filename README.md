# MenuGo-app
Aplicación móvil para estudiantes de la UCEVA que muestra restaurantes cercanos con menús económicos del día.

**Integrantes:** Juan Fernando Vargas & Manuel Alejandro Tabares

---

## Reglas de Colaboración
- **Rama principal:** `main` — siempre contiene código estable. Protegida contra push directo.
- **Ramas de feature:** se crean desde `main` con el formato `feature/nombre-descriptivo`
- Ejemplo: `feature/listado-restaurantes`
- **Ramas de fix:** formato `fix/nombre-del-bug`
- Ejemplo: `fix/gps-permission-denied`
- **Pull Requests:** todo cambio a `main` requiere abrir un PR con mínimo **1 revisión aprobada**.
- **Commits:** mensajes en español con la convención `tipo: descripción`
- Ejemplo: `feat: agrega pantalla de detalle restaurante`
